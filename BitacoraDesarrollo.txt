-----------------------------------------Problema con credenciales de acceso-------------------------------------
Para solucionar se ejecutaron estos comandos desde el gitbash:

git remote set-url origin "https://github-username@github.com/sebassanchez00/PlataformaUniversidad-C3G48-G5.git"

git push --set-upstream origin CrearPaginaNoticias_SSanchez

Luego aparece cuadro de dialogo invitando a poner el token o loguearse con el browser. Se le dio en bowser y el
sistema se logueo automático

Algunas veces no aparece el diálogo, entonces hay que ir a panel de control y borrar las credenciales almacenadas
para que las vuelva a preguntar en: Control Panel\User Accounts\Credential Manager. Se borro la que dice
git:https://github.com
